> [!IMPORTANT]
> The diff files in this folder are only valid for the (sub) commands as defined in **wado 0.2.0**.
> The build process defined in [`build.rs`](../build.rs)) generates shell completions for various shells.
> The shell completions for Fish, Bash and Zsh are patched with these diff files.
>
> If any of the parameters and options change, the diff files will most likely no longer work. In that case they need to
> be manually adjusted.
